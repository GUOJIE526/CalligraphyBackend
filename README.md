# 🖋️ 書法作品後台管理系統（ASP.NET Core MVC）

本專案為書法家個人線上展覽平台的**後台管理系統**，使用 ASP.NET Core MVC 框架開發，供書法家上傳、管理、修改作品內容。

---

## 🔧 技術架構

- ASP.NET Core MVC 8
- Entity Framework Core（DataBase First）
- MSSQL 資料庫（GUID 作為主鍵）
- Bootstrap 5 + jQuery + DataTables 前端框架

---

## ✅ 系統功能

- ✅ 書法作品新增 / 編輯 / 刪除
- ✅ 上傳與預覽圖片
- ✅ 分頁查詢、標題搜尋、排序
- ✅ 欄位驗證（含前後端）
- ✅ 自動填入建立者、修改者、IP 來源
- ✅ 可與 Nuxt 前台共享圖片資料夾
